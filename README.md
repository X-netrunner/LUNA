# LUNA
LUNA is an ai assistant for linux , coded wrt arch linux specifically 

# DEPENDENCIES
- Ollama 
- Python and pip (or pyenv)
- chromadb ,sentence-transformers ( only supports python 3.12 or 3.11 for now , So downgrade python 3.14 to 3.12)
- Models for LUNA 1.5b , 3b , 7b or 8b of your choice but change the luna.sh script accordingly

# INSTALLATION
- Unzip the folder
- place the unzipped folder into /home/usr/.local/bin (optional)
- run 'chmod +x luna.sh' in the folder
- start with : './luna.sh prompt'
